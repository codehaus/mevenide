MavenBuilder enables the activation of the action by calling the resourceDelta.accept(actionActivator)
When ActionActivator visits an IResourceDelta, for each ActionDefinition whose pattern match the path of the Delta, it enables the actionDefinition.
CustomMavenLaunchManager just fills the submenu with the ManageActionDefinitionsAction and all activated PatternBasedMavenLMaunchActions.
ActionDefinitionsManager acts as a registry of existing ActionDefinitions.
